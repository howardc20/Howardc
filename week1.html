Week 1 | Setting Up Website

The assignment for the week was to create an enclosure for an Arduino Uno or Raspberry Pi (raster, vector, 2D, 3D, render, animate, simulate, ...) and document it on my Class Page using Fusion360. I included pictures and original files on my page. I mentioned what worked and did not work when creating the enclosure. My first shape was rough. I was not comfortable with the program yet and I was still getting familiar with the tools. As I continued to create more shapes, my skills progressed.
